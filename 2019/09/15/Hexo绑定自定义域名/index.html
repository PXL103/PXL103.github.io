绑定自定义域名

之前我已经利用 Hexo+Github 搭建好了我的博客，可以正常写文发布。但是心里比较膈应的是域名，域名的后缀是 github.io ，这是由 Github 提供的二级域名，不好记，也不牛逼，综合考虑， 我还是想买了一个性化的域名，然后解析到 Github上去。

域名购买

域名可以在万网，西部数码等购买，我是在西部数码上购买的，其实当时是准备在阿里的万网上购买的，繁琐的注册，最后需要审核，我……

我性子急，不愿意等几天，所以就去西部数码买了，不用审核，第一年 10RMB，我买了3年的，花了 66RBM。也有一些是免费的域名，免费期最多可以达到12个月（第二年开始收费……），考虑到我想长期经营我的博客 (主要是我喜欢的域名都不免费)，所以我就直接买了。
